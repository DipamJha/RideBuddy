# 🚗 RideBuddy

RideBuddy is a ride-sharing web platform that helps people traveling to the same destination
share rides, reduce traffic, and save costs.

## 🌟 Features (MVP)
- Create and search rides
- Join rides with available seats
- Telegram notifications for new rides
- Smart alert subscription when no rides are available

## 🛠 Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Notifications: Telegram Bot API
- Maps: Google Maps API
